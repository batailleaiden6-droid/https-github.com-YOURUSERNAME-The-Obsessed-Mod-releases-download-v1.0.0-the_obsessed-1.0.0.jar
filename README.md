# The Obsessed Mod

**Minecraft Forge 1.21.x Mod**  
Obsessed-style psychological horror mob with GeckoLib 3D model.

---

## Features

- Stalker mob that watches the player from a distance  
- Vanishes when looked at  
- Hugs the player when close (causes slow + regen effects)  
- Kills mobs that target the player  
- Spawns naturally in dark areas/night  
- GeckoLib 3D model and animation support  
- Creepy sounds: whispers and breathing  

---

## Installation

1. Install **Minecraft Forge 1.21.x**  
2. Download the mod JAR from [Releases](https://github.com/YOURUSERNAME/The-Obsessed-Mod/releases)  
3. Place the JAR in your `mods` folder  
4. Run Minecraft with Forge  

---

## Development

- Built with **Forge** and **GeckoLib**  
- Java 17+ required  
- Resources: `src/main/resources/assets/the_obsessed`
